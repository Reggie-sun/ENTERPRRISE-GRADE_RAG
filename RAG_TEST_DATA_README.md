# RAG 测试数据说明

本文档介绍为 RAG 系统创建的测试数据及其使用方法。

---

## 1. 数据来源

测试数据基于 `造信息用的.md` 文件中的 CNC 机床 PLC/PMC 报警码翻译文件（TS 格式），原始数据包含 **200+ 条**机床报警码及其处理说明。

---

## 2. 生成的测试文档

### 2.1 CNC 机床 PLC 报警码处理手册

**文件路径**: [data/uploads/cnc_plc_alarm_manual.md](data/uploads/cnc_plc_alarm_manual.md)

**文档结构**:
- **一、急停类报警** (700000-700015)
- **二、冷却与润滑系统报警** (700004-700007)
- **三、A轴转台报警** (700008-700013)
- **四、B轴转台报警** (700016-700021)
- **五、液压系统报警** (700024-700027)
- **六、主轴系统报警** (700100-700129)
- **七、刀库与换刀系统报警** (700124-700163, 700232-700263)
- **八、测头与对刀仪报警** (700109-700115, 700337-700339)
- **九、安全系统报警** (700048-700061, 700246)
- **十、冷却水箱报警** (700028-700058, 700244-700245)
- **十一、碰撞保护报警** (700201-700202, 700254-700256)
- **十二、其他报警**
- **附录：报警码快速查询表**

**数据特点**:
- 每个报警码包含：报警码、信号地址、报警描述、处理步骤
- 处理步骤详细，便于 RAG 检索和问答
- 按系统分类组织，便于语义检索

## 3. 使用方法

### 3.1 上传文档

1. 启动 RAG 系统（参考 [DEBUG_GUIDE.md](DEBUG_GUIDE.md)）
2. 打开前端页面 http://127.0.0.1:3000
3. 上传 `data/uploads/cnc_plc_alarm_manual.md`

### 3.2 测试问答

上传完成后，可以尝试以下问题：

```
# 急停类问题
- 面板急停报警怎么处理？
- 刀库急停报警如何解决？

# 温度类问题
- A轴高温报警怎么处理？
- 主轴轴承温度过高怎么办？
- 液压站高温报警的处理方法

# 刀库类问题
- 换刀超时怎么解决？
- 刀库旋转位置错误如何处理？
- 刀具磨损报警怎么办？

# 润滑类问题
- 润滑压力不足怎么处理？
- 油气润滑故障如何解决？

# 冷却类问题
- 水冷机故障怎么处理？
- 水箱液位低怎么办？
- 切削液浓度报警怎么解决？

# 安全类问题
- 安全门未关闭怎么办？
- 天窗未关闭如何处理？

# 复杂问题
- 700102报警是什么意思？
- B轴转台泄漏怎么处理？
- 主轴松拉刀信号异常如何解决？
```
